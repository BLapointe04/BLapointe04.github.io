CSCE 242 - Web Applications

The web applications class will teach us to build functional websites using HTML, CSS, and more. We'll learn modern techniques like source control, web development, and server-side applications.

View My GitHub Repository

Assignments

Assignment 1: Basic HTML

Built a basic HTML page using headings, lists, links, and semantic structure to understand page layouts.

Assignment 2: Basic CSS

Learned to style pages using an external stylesheet, experimenting with colors, fonts, spacing, and borders.

Assignment 3: Page Layout

Applied Flexbox to build responsive multi-column layouts, matching the provided wireframe diagrams.
